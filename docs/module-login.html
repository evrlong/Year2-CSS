Module: login

Login functionality for the application. This module handles user login, token management, and redirection after successful login. It also includes validation for the login form inputs.

Methods

(inner) login(url, data) → {Promise.<object>}

This function sends a POST request to the specified URL with the login data. It handles the response, stores the access token in local storage, and redirects the user to their profile page.
Parameters:
Name Type Description
url string The URL to send the login request to.
data object The login data containing email and password.
